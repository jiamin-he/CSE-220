Thank you for using Web Historian, the extension that visualizes your web use to help you understand your habits.

Creating your visualizations…


Right-click for many important features.

This animated GIF shows one of the many right-click features of Web Historian, going from the Web Visits view to the Time Heatmap view when you want to see when you visited a site. Try right-clicking in all of the visualizations to see many useful options. Often a right-click is a two-finger-tap on a trackpad. Check out other Web Historian features on our website.

For more information about Web Historian visit http://webhistorian.org.
